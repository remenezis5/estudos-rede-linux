Dia 1 (12/02/2026) – Introdução à Rede

Hoje comecei o curso.

Aprendi que:

TI (Tecnologia da Informação) é o uso de tecnologia digital para armazenar,
processar e organizar dados para gerar informações úteis.

A área de TI é ampla e envolve diferentes profissionais, como:

- Engenheiros de rede, que garantem a comunicação entre computadores.
- Técnicos de hardware, que substituem e reparam componentes.
- Equipe de suporte, que ajuda usuários a resolver problemas.

Entendi que TI não é só programação.
É uma área grande, com várias especializações.
